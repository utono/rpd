# Hammerspoon Keybindings

All keybindings use Apple virtual keycodes for RPD layout compatibility.

| Shortcut | Keycode | Action |
|----------|---------|--------|
| Cmd+Space | 49 | Toggle two most recent windows of same app |
| Cmd+o (physical S) | 1 | Toggle two most recent windows of same app |
| Cmd+Option+Space | 49 | Cycle all windows of focused app |
| Cmd+Option+o (physical S) | 1 | Cycle all windows of focused app |
| Cmd+Return | 36 | Open new kitty instance |
| Cmd+\| (physical =) | 24 | Screenshot |
